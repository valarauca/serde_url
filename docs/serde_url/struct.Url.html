pub struct Url { /* fields omitted */ }

Opaque type that can be serialized/deserialized and acts like a string.

The goal is not URL maniplutation, but rather reading and writing URL's, as well as ensuring a consistent format.

Note Usernames/Passwords

Non-UTF8 usernames and passwords will generate an error.

Note Hashing & Equality

This type implements std::hash::Hash, it will use the output of get_string() for the purposes of hashing or comparison. Either as a utf8 string, or array of bytes.
